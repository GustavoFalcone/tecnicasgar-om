# Entregável Garçom Profissional

Projeto independente da landing page para a área de membros do produto “+200 Técnicas para Servir como um Garçom Profissional”.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
```

Este diretório possui seu próprio repositório Git, dependências, configuração do Vite, build e configuração de deploy. Ele pode ser publicado em um repositório e domínio totalmente diferentes da landing page.
