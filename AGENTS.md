# Produto

+200 Técnicas para Servir como um Garçom Profissional

# Direção visual

- Manual técnico visual premium.
- Forte presença de imagens demonstrativas.
- Pouco texto por página.
- Compreensão rápida.
- Fonte Poppins.
- Branco `#FFFFFF`.
- Preto carvão `#171717`.
- Dourado champanhe `#C6A15B`.
- Ilustrações e fotografias sem textos incorporados.
- Textos, setas, círculos e legendas devem ser adicionados pelo código.
- Imagens maiores do que os blocos de texto.
- Variedade real de composição.

# Evitar

- Páginas formadas apenas por cards.
- Mesmo layout em todas as técnicas.
- Grandes parágrafos.
- Ícones genéricos como visual principal.
- Visual escolar ou infantil.
- Gradientes coloridos.
- Cores fora da paleta.
- Bonecos geométricos genéricos.
- Malabarismos perigosos.
- Carregamento exagerado de pratos.
- Práticas sem higiene ou segurança.

# Fluxo obrigatório

1. Prototipar primeiro.
2. Aplicar em poucas páginas.
3. Testar visualmente.
4. Somente depois escalar.
5. Executar `npm run build` após cada fase relevante.
